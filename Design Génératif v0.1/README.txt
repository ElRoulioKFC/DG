Veillez suivre les instructions suivantes:


1/ Veuillez réaliser le shield pour la carte arduino, 
   le montage se trouve "shéma shield v0.1.png"

2/ Veuillez téléverser le code de l'arduino dans l'arduino,
   le code se trouve dans "shield v0.1.ino"

3/ Veuillez lancer le programme,
   l'executable qui s'appelle launch



Il est possible que la connection entre l'arduino ne marche pas, vérifier que l'arduino est branché au port [1]

[0] "/dev/cu.Bluetooth-Incoming-Port"
[1] "/dev/cu.usbmodem14101"
[2] "/dev/tty.Bluetooth-Incoming-Port"
[3] "/dev/tty.usbmodem14101"

Si ce n'est pas le cas contactez nous pour qu'on règle le problème
